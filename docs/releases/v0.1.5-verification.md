version was executed through Go:

```powershell

$issuer = 'https://token.actions.githubusercontent.com'

go run github.com/sigstore/cosign/v3/cmd/cosign@v3.1.1 verify `
  --certificate-identity-regexp $identity `
  --certificate-oidc-issuer $issuer `
  --output json `

  --certificate-identity-regexp $identity `
  --certificate-oidc-issuer $issuer `
  --output json `

```
