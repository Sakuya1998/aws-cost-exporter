# v0.1.0 release checklist

## Automated gates

- [ ] Linux CI green: tests, race, 75% coverage, assets, container
- [ ] `./test/perf/...` passes (series budget, API budget, scrape baseline)
- [ ] `./test/release/...` passes (GoReleaser + release workflow)
- [ ] `goreleaser check` succeeds; staging tag release reviewed

## Runtime contracts

- [ ] Scrape does not call AWS (`test/perf/gate_test.go`)
- [ ] Startup refresh = 8 `GetCostAndUsage` + 1 `GetCostForecast` when unpaginated
- [ ] Paginated deployments: perf API-budget gate (8+1) does not apply; verify `pagination_pages_total` and alert thresholds instead
- [ ] `max_pages=50` hard-stops paginated queries; `series_limit` caps export only
- [ ] `series_limit=1000` conserves totals; 1000-series scrape < 15s
- [ ] `/ready` matches cache freshness; currency-safe metrics; no cartesian grouping
- [ ] README/IAM/dashboard/rules/chart/config compatibility tests pass

## Supply chain

- [ ] Signed GHCR image + Helm OCI chart; `cosign verify`; Trivy clean
- [ ] Chart `appVersion`/image tag match SemVer; only stable tags update `latest`
- [ ] Multi-arch build verified (`test/container`)

## Manual sign-off

- [ ] MVP IAM on real read-only account; release notes reviewed
- [ ] arm64 runtime smoke (`docker run --platform linux/arm64`) when available

See `ROADMAP.md`, `SECURITY.md`, `docs/operations/troubleshooting.md`.
